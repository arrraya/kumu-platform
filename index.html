Architecture

Technical Stack

⚡
FastAPI + Python
Async REST backend with auto OpenAPI/Swagger docs. Chosen for performance, static typing, and collaborative dev.
⚛
React + TypeScript
Type-safe frontend with Redux Toolkit for state, D3.js for complex visualizations, and PDF export.
🐘
PostgreSQL
Normalized schema with materialized views, optimized indexes, and triggers for derived data consistency.
🤖
XGBoost ML
Prediction service trained on 10K–50K records. Outperformed Random Forest and neural nets on this dataset.
🔴
Redis Cache
Caches computationally intensive analytics results, reducing server load by 70% on frequent queries.
🐳
Docker Compose
Containerized local deployment with automated setup scripts, DB migrations, and seed data.
